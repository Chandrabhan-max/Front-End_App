# 🚀 Billing & Inventory Management System

A modern, full-stack **Billing & Inventory Management** application designed for businesses to manage customers, track inventory, and generate professional invoices instantly.

---

## ✨ Key Features

* **Comprehensive Dashboard:** Real-time analytics including Total Revenue, Today's Invoices, and Monthly Growth.
* **Customer CRM:** Create, update, and manage customer profiles with Active/Inactive status tracking.
* **Inventory Control:** Manage items with unique Item Codes, selling prices, and real-time availability.
* **Dynamic Invoicing:** Automated invoice generation with smart GST application based on customer registration status.
* **Bulk Processing:** Upload CSV files to generate hundreds of invoices in a single click.
* **Export Options:** High-quality **Print** functionality and **Instant PDF Download** for all invoices.
* **Responsive UI:** Optimized for a seamless experience across Desktops, Tablets, and Mobile devices.

---

## 🛠️ Tech Stack

**Frontend:**
* React.js (Vite)
* Material UI (MUI)
* React Router DOM
* Axios (API Management)
* jspdf & html2canvas (PDF Engine)

**Backend:**
* Node.js & Express
* PostgreSQL (Database)
* XLSX (Excel/CSV Processor)
* fs-extra (File Management)
